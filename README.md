## 🎵 프로젝트 설명

- EZ to Play는 누구나 아티스트가 될 수 있는 공연 예약 플랫폼입니다.

- 아티스트는 자신의 공연을 등록하여 홍보할 수 있습니다.

- 사용자는 공연 목록을 확인하고 마음에 드는 공연을 예약할 수 있습니다.

## 🎵 구현 과정

### 1주차
- 아이디어 기획
- 디자인 완성

### 2주차
- 공통 컴포넌트 작업
- 회원가입 로직 구현
- 로그인 로직 구현
- API 요청 테스트 시작

### 3주차
- 아티스트 등록 및 수정 구현
- 공연 등록, 수정, 삭제 구현

### 4주차
- 공연 예약 구현
- 후기 등록, 수정, 삭제 구현

### 프로젝트 기간 이후
- 피드백 반영
- 버그 수정(수시)

<br>

## 🎵 주요 기능

- 메인 페이지

- 회원가입 및 로그인

- 아티스트 등록 및 정보 수정

- 마이 페이지

- 아티스트 정보 조회

- 공연 등록 및 수정

- 공연 정보 조회

- 공연 예약

- 후기 작성

### 추가 예정 기능

- OAuth 로그인

- 공연 예약 시 결제

- 지도상에 여러 개의 공연 장소를 표시하는 기능
